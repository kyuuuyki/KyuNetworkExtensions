## Requirements

- iOS 13.0+
- Xcode 11.0+
- Swift 5+

## Installation

KyuNetworkExtensions is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'KyuNetworkExtensions'
```
